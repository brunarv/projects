In this homework I've deployed the .slideDown()effect that I found on JQuery website.

Every time that someone clicks on CLICK HERE button, the next section will be displayed or it will be hidden. 

				
$( "#section" ).hide();

$( "#clickme" ).click(function () {
	if ( $( "#section" ).is( ":hidden" ) ) {
		$( "#section" ).slideDown( "slow" );
	} else {
	$( "#section" ).hide();
	}
});

To check how it works, you just need to click on the button bellow.
